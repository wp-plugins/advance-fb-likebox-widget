== Description ==
Advance facebook like box enables you to have a facebook like box over your Wordpress website all or any pages. Equipped with the advance features it is equally easy to customize and integrate with your website. Advance facebook like box gives you the direct like button along with the list of users with their images who have already liked your facebook page. For the implementation of the like box, the admin could easily select the pages and posts they wish to show it on. 



 

== Features ==

* You can display it on the landing page of the website.
 
* You may exclude the landing page and show it on all the inner pages.

* You can easily display it on any default post or custom posts of the website.



== Installation ==

= Install =

1. Login into your WordPress admin panel
2. Navigate to Plugins => Add New
3. Click Upload 
4. Click Choose File and select the advanve-fb-likebox-widget.zip.
5. Click Install Now.
6. Activate the plugin through the 'Plugins' menu in WordPress.
7. Go to Dashboard => appearnce => widget => Drag & drop Advance Fb like box wdget to your specific sidebar.




= Uninstall =

1. Deactivate FB like box widget in the 'Plugins' menu in Wordpress.
2. After Deactivation a 'Delete' link appears below the plugin name, follow the link and confim with 'Yes, Delete these files'.
3. This will delete all the plugin files from the server as well as erasing all options the plugin has stored in the database.
